Endcraft

EndCraft is a Minecraft server I've owned with some friends since 2013. I've been working on and off developing and running the server over that time, which really doesn't yeild results. So, I decided to create this repository; a place to leave public files and archives of EndCraft to help people with more drive than me develop other servers. EndCraft was originally developed for MC:PE in 2013 using a mix of vanilla and PMMP throughought its first few years. The server moved to being vanilla-based in 2017, coming back to PMMP in 2019. EndCraft also was ported to Java in 2019 using Paper. In early August 2020 I shut down Java for the last time as of writing and left the team maintaining MC:BE. A name change and few months later MC:BE EndCraft, known as Helios Practice, closed on Febuary 1st, 2021.

Being the main active theorizer of all Freebird games, I noticed theories were hard to keep track of and I would usually lose ones made by others. Custom Query Stats

Custom Query Stats is a plugin that stores player statistics in your server's MOTD. These stats are hidden from players but visible to query scripts. In many ways, Custom Query Stats is the precursor to Webcore, and includes some of Webcore's features. Unlike Webcore, however, Custom Query Stats uses the server MOTD to transfer data rather than HTTP. Custom Query Stats is limited in both version (1.16.x) and information (2MB worth of player statistics), so I'd recommend using Webcore's API for better results. If you run a small server and need a lightweight, simple data transfer plugin however, I'd recommend Custom Query Stats.
