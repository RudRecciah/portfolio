Welcome

Welcome to EndCraft!

About

EndCraft is a Minecraft server I've owned for about # years. I've been working on and off developing and running the server over that time, which really doesn't yeild results. So, I decided to create this website; a place to leave public files and archives of EndCraft to help people develop other servers.

Past Development

EndCraft was originally developed for MC:PE in 2013 using PMMP. Moving to MC:BE in 2018 and Java Edition in June 2019, MC:BE builds were vanilla (it's suprisingly decent) and Java Edition builds used Spigot. In April 2020, MC:BE moved back to PMMP and Java Edition to Paper. MC:PE and BE builds used the latest stable versions of the game, while Java Edition used a mix between 1.15.x and 1.8.x. Custom plugins for Java Edition used the 1.13 Spigot API in both Spigot and Paper variants. I've left the team developing MC:BE builds, and at the time of writing they are maintaining heavily modified versions of EndCraft's MC:BE builds. Public versions of EndCraft's Java Edition builds are no longer maintained as of writing, following the trend of previous years. Some parts of those builds are available here.

PMMP Archive

You can find information about EndCraft's PMMP builds during my time developing them here. As the site is an archive, links may become broken over time and new information will not be added. You will not be able to find any of the builds themselves there though; they are all closed source and will most likely never be released.

Current and Future Development

EndCraft, while not currently being developed by me, has parts of itself implemented in other servers (most notably the modified EndCraft PMMP builds from above). I currently don't have any plans to develop EndCraft in the future, but if you want to get in touch about potentially rebooting it in the future don't hesitate to contact me. I'm open to potentially going into development with a new team or supplying someone I can trust with closed source parts of EndCraft to develop or maintain them.
