DEVELOPMENT

Developers at EndCraft do much of the work to keep the server running. We code practically everything ourselves, so it's necessary to have a few developers who can work on maintence or adding new feautres. To become a developer, we recommend that you have some knowledge with PHP, specifically a good understanding of functions and the PMMP api. Other languages you could code in are HTML, CSS, and JS, but we need more PHP developers then any other language. If you want to help with some networking aspcets, we recomemnd having a good amount of MySQL knowledge and have a good understanding of how networking works.

APPLY

Becoming a developer is not that easy of a task as you can't offically apply for a development position. We recommend contacting an owner on Discord to become a developer. We prefer people who are already trustworthy staff to apply for development, so if you aren't staff already it would be worth your time to apply. Networking positions are harder to get into, and we rarely consider accepting people that are not already in our development or staff teams.
